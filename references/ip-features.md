# Fun Fun Pigeon IP 特征说明

## 核心外形（必须严格还原）

| 特征 | 描述 |
|------|------|
| 身体 | 圆滚滚的水滴形，紫色（#9B6FCF 左右的薰衣草紫），有轻微水彩晕染深浅变化 |
| 描边 | 棕色粗线条，类似手绘马克笔笔触，略有不规则感 |
| 耳机 | 头顶棕色弧形细头带，两侧蓝色毛绒耳机（像蓬松棉花球，#7EC8E3 天蓝色） |
| 嘴巴 | 黄色小三角形，尖朝下，位于脸部中下方 |
| 眼睛 | 两个小黑圆点，靠近中间，表情靠眼形变化 |
| 翅膀 | 两侧各一个小翅膀，贴在身体两侧，圆润短小 |
| 脚 | 非常短小，几乎隐藏在身体下方，偶尔可见 |
| 质感 | 整体软糯，水彩+马克笔混合，有轻微纹理感 |

## 表情特征细节

| 表情 | 眼睛变化 | 额外元素 |
|------|---------|---------|
| 开心 | 弯月形（笑眼） | 无 |
| 眯眼 | 细缝形 | 嘴角微翘 |
| 生气 | 八字眉压低 | 头顶爆炸线/火花符号 |
| 惊讶 | 圆眼放大 | 头顶两个!! |
| 害羞 | 正常小圆点 | 两侧粉色腮红圆圈 |
| 困倦 | 半闭（上眼皮下垂） | 头顶 ZZZ |
| 流泪 | 正常+泪珠 | 眼角蓝色水滴泪珠，2-3滴 |
| 思考 | 正常 | 头顶? ，身体略微歪斜 |

## 画风关键词（提示词必须包含）

- 日系治愈卡通插画
- 马克笔线稿 + 水彩晕染
- 棕色粗描边
- 莫兰迪奶油色系
- 软萌圆润
- 类似 Chiikawa / Sumikko Gurashi 的质感
- 干净留白，不过度堆砌元素
