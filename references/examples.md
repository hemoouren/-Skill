# Fun Fun Pigeon 提示词示例库

## 示例 1：「今天很累」

**创作说明：**
选用「困倦」表情匹配疲惫感；道具用热饮强化休息氛围；配色选粉紫夜晚色，温柔包裹感。

```
【IP 角色】
Fun Fun Pigeon 小紫鸽，圆滚滚的水滴形身体，
薰衣草紫色身体（紫色有轻微水彩晕染深浅变化），
头戴蓝色毛绒耳机（两侧天蓝色蓬松棉球耳机，顶部棕色细弧形头带），
黄色小三角嘴（尖朝下），黑色两个小圆点眼睛（困倦表情：上眼皮下垂半闭，头顶有 ZZZ 符号），
两侧各一个圆润短小翅膀贴身，脚极短几乎隐藏，
整体像一颗软糯的紫色圆润水滴，体型饱满可爱。

【画面风格】
韩国治愈插画风，儿童绘本风，手绘涂鸦感，
蜡笔颗粒描边，油画棒质感，粉彩笔涂抹效果，
毛茸茸边缘，棕色粗线手绘描边，
低细节圆润造型，纸张纹理，手工绘制感，
柔和光影，极简背景，
马卡龙配色，莫兰迪奶油色系，
整体氛围温暖陪伴，小红书爆款治愈插画风格，
类似 Chiikawa / Sumikko Gurashi 的软萌质感。

【画面内容】
背景：纯色平涂，粉紫色（莫兰迪色系），
画面主体是小紫鸽，困倦表情，双手捧着一个冒热气的白色杯子（杯身有爱心图案），
背景纯色，不添加任何装饰元素，角色之外大量留白，
左下角有手写体小字：Fun Fun Pigeon♡

【文案】
画面中有手写风格文字：「累了就好好休息一下吧～」
文字颜色深棕色，字体圆润可爱，清晰可读。

【技术参数】
竖版构图，比例 3:4，
不要出现多余人物，不要出现中文以外的文字，
整体风格：温柔治愈，像一张手绘小卡片。
```
